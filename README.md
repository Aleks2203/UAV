# UAV

## Overview

This project is a fixed-wing UAV designed and built as a personal engineering project.

The goals of the project are:
- Learn aerodynamics
- Learn flight control systems
- Develop CAD and manufacturing skills
- Build a complete working UAV

## Features

- Custom airframe
- Radio control operation
- Telemetry system
- Flight controller integration
- Modular electronics layout

## Hardware
- Receiver: FS-IA6B
- Transmitter: FlySky FS-I6
- Motor: BetaFPV 1550KV 
- ESC: 40A Unbranded
- Battery: 3S 5000mAh lipo
- Servos: FS90
